# 動滋券店家資料爬蟲

這個爬蟲會自動爬取 500.gov.tw 網站上臺北市的動滋券合作店家資料，並輸出為 CSV 檔案。

## 安裝

```bash
cd scraper
npm install
```

## 使用方法

```bash
npm start
# 或
node scraper.js
```

## 輸出

爬蟲會生成 `vendors.csv` 檔案，包含以下欄位：

- 商家名稱
- 地址
- 營業電話
- 營業時間
- 網址
- 描述
- 類別

## 注意事項

1. 爬蟲使用 Puppeteer 模擬瀏覽器，首次執行會下載 Chromium
2. 預設為非無頭模式（`headless: false`），方便觀察爬蟲執行過程
3. 爬蟲會自動處理分頁，逐頁爬取所有店家資料
4. 每個店家會開啟新分頁訪問詳情頁，提取完整資訊
5. 建議在網路穩定的環境下執行，避免中途中斷

## 執行時間

根據店家數量，完整爬取可能需要 30 分鐘到 2 小時不等。

## 疑難排解

如果遇到問題：

1. **找不到元素**：網站結構可能已變更，需要更新選擇器
2. **超時錯誤**：網路連線不穩定，可增加 timeout 時間
3. **記憶體不足**：可調整 `headless: true` 減少資源使用
